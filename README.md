# Portfolio Instructions

## How to View
Simply open `index.html` in your web browser (Chrome, Edge, Firefox, etc.).

## Customization
1.  **Images**: Replace the images in `assets/images/` with your own `profile.jpg` and `project.jpg`.
2.  **CV**: Replace `assets/cv.pdf` with your actual CV file.
3.  **Text**: Edit `index.html` to change the text content.
4.  **Links**: Update the `href` attributes in the Social Links section of `index.html`.

## Technologies Used
-   **HTML5**
-   **Tailwind CSS** (via CDN)
-   **Custom CSS** (Glassmorphism)
-   **JavaScript** (Scroll animations)
-   **FontAwesome** (Icons)
